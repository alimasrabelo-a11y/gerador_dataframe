DataGen: Gerador de Big Data

Script em Python para geração de dados sintéticos de e-commerce, ideal para praticar manipulação de grandes volumes de dados (ETL) e análise com Pandas/Spark.
📊 Estrutura do CSV

O arquivo dados_transacionais.csv contém:

    IDs: Transação, Cliente e Produto.

    Financeiro: Valor fixo por produto (formato R$).

    Temporal: Data e Hora aleatórias (2013-2026).

    Atividade: Compra, Visualização ou Carrinho.

⚙️ Tecnologias

    Python 3

    Pandas & Numpy

🚀 Como usar

    Execute o script no Google Colab.

    Altere o valor em aleatorio(10000) para a quantidade de linhas desejada.

    Responda S ao final para baixar o arquivo.

💡 Objetivo de Estudo

Simular cenários reais de Big Data para praticar limpeza de dados, conversão de tipos (strings para datetime) e cálculos de conversão de vendas.
